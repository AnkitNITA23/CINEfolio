# **App Name**: CineFolio

## Core Features:

- Home Page Carousel: Display a rotating carousel of featured movies/shows with background video/image, title, genres, ratings, and action buttons.
- Movie/Series Details: Show detailed information about a selected movie or series, including a large hero banner with a parallax effect, metadata, ratings, trailer, cast & crew carousel, and user actions (Watchlist, Watched, Like/Dislike).
- Intelligent Recommendations: Personalize movie/series recommendations based on viewing history and liked titles using AI, providing dynamic 'Because you watched...' sections.
- Advanced Search & Discovery: Implement a smart search bar with live suggestions, filters (genre, rating, language, year), lazy loading, infinite scroll, and sorting options for discovering new content.
- User Profile Dashboard: Create a user profile dashboard to display user avatar, stats (movies watched, watchlist count, favorite genres), and tabs for Watchlist, Watched History, Liked Titles, and Recommendations.
- Authentication: Implement login/signup pages with a modern UI, animated form transitions, and placeholders for social login.
- Personalized Recommendation Tool: Employ an AI tool to enhance user content discovery by analyzing viewing patterns, watchlists, and preferences. The AI tool will reason when or if a title is likely to be of interest to the user and feature that title in the 'Recommended for You' section.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey a premium and cinematic feel, avoiding cliches by steering clear of predictable colors often associated with entertainment platforms. Based on the concept of premium cinematic experiences.
- Background color: Dark gray (#212121), desaturated and dark, to ensure comfortable viewing in dark mode.
- Accent color: Electric indigo (#6F00FF), analogous to deep purple, for highlights and interactive elements, creating a contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, suitable for both headlines and body text.
- Use a set of minimalist icons with subtle line weights, custom-designed to suit the cinematic theme.
- Implement a responsive grid system with professional spacing to maintain consistency across all devices.
- Employ smooth micro-animations and page transitions to enhance the user experience, including fade/slide transitions, card hover zoom, and button ripple effects.
